DJANGO APP

He desarrollado un gestor de tareas utilizando el framework Django, que permite a los usuarios gestionar sus tareas de manera eficiente, esta aplicacion la utilizare como plantilla para un mejor gestor y mejoras de UI y UX, esto son solo los avances. Para acceder a las funcionalidades del gestor, los usuarios deben registrarse y crear una cuenta. Solo aquellos que están autenticados pueden ingresar a las rutas /tasks y /tasks/create, lo que garantiza que cada usuario pueda gestionar sus propias tareas de manera segura.

El sistema ofrece la capacidad de listar, agregar, actualizar y eliminar tareas, cubriendo así lo esencial de un CRUD. Los usuarios pueden ver todas las tareas que han creado, así como crear nuevas tareas mediante un formulario donde pueden ingresar detalles como el título y la descripción. También tienen la opción de modificar los detalles de tareas existentes o eliminarlas si ya no son necesarias. La experiencia de usuario es intuitiva y fluida, lo que facilita la gestión de tareas.

En cuanto a la base de datos, se utiliza MySQL como sistema de gestión, lo que permite almacenar de manera eficiente la información de las tareas y los usuarios. La integración con Django se realiza a través de modelos que simplifican la interacción con la base de datos, permitiendo realizar consultas y manipulaciones de datos de manera sencilla.

La interfaz de usuario está diseñada para ser clara y accesible, utilizando Bootstrap para asegurar que sea moderna y responsive, adaptándose a diferentes dispositivos y tamaños de pantalla.

Este proyecto no solo me ha permitido practicar con Django, sino también familiarizarme con el uso de bases de datos y la implementación de características de seguridad en aplicaciones web.
